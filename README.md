# 🍔 Java 點餐系統 (Restaurant Ordering System)

這是一個使用 Java 與 Swing 開發的桌面端圖形化點餐系統 (GUI)。
使用者可以透過介面進行單點、選擇套餐，系統會自動計算總金額，並產生詳細的結帳收據。

## ✨ 功能特色 (Features)
* **圖形化介面 (GUI)**：使用 Java Swing 打造，提供直覺的點餐面板。
* **動態讀取菜單**：透過讀取外部 CSV 檔案 (`Single_meal.csv`) 來載入單點菜單，方便隨時修改餐點與價格。
* **單點與套餐組合**：支援獨立單點項目，以及預設的套餐組合計算。
* **收據自動生成**：結帳完成後，會自動在專案目錄下產生 `receipt.txt` 實體收據檔案（支援 UTF-8 編碼，完美顯示中文）。

## 🛠️ 執行環境 (Prerequisites)
* Java Development Kit (JDK) 17 或以上版本
* 建議編輯器：VS Code、Eclipse 或 IntelliJ IDEA
